We use two external libraries. In the fastwht/python folder. setup.sh has to be run. 
In the aptly-proximal folder "python setup.py install" has to be run. 
In order to generate the pkl files used In the graphs the four experiment runner files named:
runSWHTVertex.py
runSWHTEdge.py
runStobbeVertex.py
runStobbeEdge.py
Have to be run.
The Jupyter notebook file PlotData2.ipynb plots the graphs used in the paper.
If you use a package manager an environment.yaml file is provided for conda.
